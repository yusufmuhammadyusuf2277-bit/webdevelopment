HTML Links

What is an HTML Link?

HTML links allow users to move from one page to another. Links are created using the anchor (<a>) tag.

Example of a Link

<a href="https://yusufmtech.com">Visit YusufMTech</a>

Opening Links in a New Tab

<a href="https://yusufmtech.com" target="_blank">
  Open in new tab
</a>

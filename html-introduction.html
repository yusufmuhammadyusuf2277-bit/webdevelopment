What is HTML?

HTML stands for HyperText Markup Language. It is used to create the structure of web pages and web applications.

Why Learn HTML?

  • It is the foundation of web development
  • Easy to learn
  • Works with CSS and JavaScript

Simple HTML Example

<!DOCTYPE html>
<html>
<body>
  <h1>Hello World</h1>
</body>
</html>
